## 📊 **Architecture**

### **🏗️ Design Pattern**
- **Modulaire**: Séparation HTML/CSS/JS
- **Responsive**: Mobile-first design
- **Accessible**: Standards WCAG
- **Maintenable**: Code documenté

1. **Saisie salle** → Validation format
